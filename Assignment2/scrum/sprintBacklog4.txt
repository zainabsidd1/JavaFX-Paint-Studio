NAME     : Undo/Redo not implementing mover tool changes
PRIORITY : 2
ESTIMATE : (1 hour, 2 hours, 4 hours, 1 day, 2 days)          ACTUAL :
ISSUE    : The current Undo/Redo logic only tracks creation and deletion
           of shapes, but not movement changes done using the Mover tool.
